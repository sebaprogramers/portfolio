RV Motos: Gestión de Stock

Sistema Full Stack con Sincronización en Tiempo Real

El Desafío

El negocio requería una herramienta centralizada para controlar el inventario de Motos, evitando errores manuales y permitiendo consultas rápidas desde cualquier dispositivo.

La Solución

Desarrollé una Web App robusta que permite la carga, edición y baja de stock. La interfaz es intuitiva y está optimizada para el flujo de trabajo diario del taller.

Tecnología Aplicada

  • JavaScript: Motor de la lógica del sistema.
  • Firebase Firestore: Base de datos NoSQL para sincronización inmediata.
  • Firebase Auth: Acceso seguro y controlado.
